# Expense Tracker

## Description
Expense Tracker is a command-line Python application that helps users record, organize, and manage their expenses. Users can add expenses, categorize them, search for specific expenses, track their budget, view spending summaries, and store data permanently using CSV files.
This project was developed while learning Python and demonstrates the use of functions, lists, dictionaries, file handling, CSV operations, testing with pytest, and error handling.

## Features
* Add new expenses
* View all expenses
* Delete expenses
* Search expenses by name
* Categorize expenses
* View category-wise spending summaries
* Calculate total expenses
* Set and track a budget
* Sort expenses
* Save expenses to a CSV file
* Load saved expenses automatically when the program starts
* Unit tests using pytest

## Technologies Used
* Python 3
* CSV Module
* Pytest
